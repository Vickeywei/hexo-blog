mac下使用hexo搭建blog

使用hexo搭建blog for sunny


写这篇文章的起因是来自一篇大神的博客,如果有什么不对希望多多指正


安装nvm


在终端Terminal中执行下面的语句


$ curl https://raw.githubusercontent.com/creationix/nvm/v0.7.0/install.sh | sh


等待不会太长时间终端会提示


=> Close and reopen your terminal to start using NVM


然后推出中端重启就完成了.因为只有重启nvm才能生效


使用nvm安装node.js


$ nvm install 0.10


安装hexo


$ npm install -g hexo


然后初始化blog的文件


然后利用终端进行初始化文件


$ cd desktop


$ mkdir blog


$ hexo init


$ ls


文件目录层级是长这样的,别眨眼请看好



新建一篇文章,我的已经建好了.看这就是!在哪?这篇文章就是!


$ hexo new 我的第一篇blog


然后编辑md文件,完成之后啦就可以在本地预览打开了,具体操作如下


$ hexo generate


$ hexo server


如果你感觉你的主题不是那么炫目夺人,那好老司机就把官网告诉你你自己去看吧.主题官网


选择好你中意的那个把他clone下来然后在配置文件中修改一下就OK了


例如


git clone git://github.com/tommy351/hexo-theme-light.git themes/light


_config.yml配置文件中设置


theme: light


如果感兴趣的同学也可以买个域名为自己搭建一个属于自己的博客地址



  • 买域名

  • 托管在github上,创建CNAME文件,里面协商最终的域名如blog.vikey.com >public/CNAME

  • 然后就是区域名后台配置



提交到github上


在github上新建远程仓库,完后再刚才配置主题的那个文件中修改一些东西
